## CV numérique 💻📄 (Avril 2025)
![Badge](https://img.shields.io/badge/github-%23181717?logo=github)
![Badge](https://img.shields.io/badge/-HTML-E34F26?style=flat&logo=html5&logoColor=white)
![Badge](https://img.shields.io/badge/-CSS-1572B6?style=flat&logo=css3&logoColor=white)

**Objectif** : Créer une page web simple qui présente votre profil personnel(CV, portfolio) ou un thème fictif (par exemple : un hobby, un produit, ou un événement) en utilisant les principaux éléments de HTML et CSS.

**Description** : Réaliser un CV numérique à peaufiner et présenter aux entreprises.
